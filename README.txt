/*------------- DESCRIPCIÓN DEL PROYECTO ---------------------*/
  
	Dr Locz es un sistema donde es posible relizar las siguientes tareas:

	Registros de Usuarios. 

  	Debates de temas publicados por especialistas en nuestro blog.

	Crear historiales y registrar nuevos sintomas en cada Usuarios para saber cuales enfermedades son mas comunes.

	Administracion de usuarios.

   
/*--------------------------INSTALACIÓN----------------------*/

   
	1. Descomprimir el archivo "drlocz.rar" que se encuentra en la carpeta "doc"
	2. Importar el archivo sql en phpmyadmin
	3. Modificar el archivo configuracion.php que se encuentra en el modulos/configuracion
	4. Crear un usuario en http://localhost/locz/     modulo "Registrarse"
	5. Ya creado el usuario te redireccionará a tu perfil para que empieses a utilizar Dr Locz
